What Triggers a Consent Receipt
Every sensitive action creates a consent receipt — an authoritative record you can keep, share, or use to appeal. Compliant with ISO/IEC TS 27560.
What a Consent Receipt Contains
Every receipt includes: action type and description / who approved it / exact timestamp and timezone / fee and cost shown / risk statement accepted / tools and connectors active during the action / reversal or appeal path / Decision Ledger ID.

Format: machine-readable JSON + plain-language card. Download anytime from Grid Account.
Consent receipts are retained for a minimum of 7 years per AUSTRAC AML/CTF requirements. Receipt issuance is logged to the Decision Ledger. Standard: ISO/IEC TS 27560.
