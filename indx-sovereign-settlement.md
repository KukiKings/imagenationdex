## 7. SIINDEX Pre-Flight Check Protocol

Before any transfer above threshold:

1. Recipient identity verification
2. Compliance and sanctions scan
3. Optimal settlement path calculation
4. Transfer risk assessment

Output: risk level (Low/Medium/High) + recommended rail + estimated fee + estimated settlement time + any required approval gates.

If high-risk: human review required before funds move.  
If cultural assets involved: Cultural Rights Graph check.  
If child citizen: guardian approval required.
